## Design Guidelines

### Color Scheme:
- **Primary:** Blue (#0066CC) - technology, trust
- **Secondary:** Green (#00CC66) - success, growth
- **Accent:** Orange (#FF6600) - energy, innovation
- **Background:** White or light gray

### Fonts:
- **Headings:** Sans-serif (Arial, Calibri, or Segoe UI)
- **Body:** Sans-serif, size 18-24pt for readability
- **Code/Technical:** Monospace (Consolas or Courier)

### Visual Tips:
- Use icons for bullet points
- Include diagrams, not just text
- Keep slides uncluttered (max 6 lines per slide)
- Use high-quality screenshots
- Maintain consistent style throughout

## Creation Tools

### Option 1: PowerPoint (Windows)
1. Open PowerPoint
2. Choose a professional template
3. Create slides based on outline above
4. Export as PDF: File → Save As → PDF

### Option 2: Google Slides (Free, Online)
1. Go to slides.google.com
2. Create new presentation
3. Build slides
4. Download as PDF: File → Download → PDF

### Option 3: Canva (Free, Easy)
1. Go to canva.com
2. Search "Presentation" templates
3. Use "Tech" or "Professional" theme
4. Export as PDF

## PDF Export Checklist

Before exporting to PDF:
- [ ] All text is readable (minimum 18pt font)
- [ ] Images are high quality
- [ ] No spelling errors
- [ ] Team name and member names correct
- [ ] All links are correct
- [ ] Slides are in logical order
- [ ] File size under 5MB (hackathon requirement)

**To reduce PDF size if needed:**
- Compress images before adding
- Use "Standard quality" export
- Remove any unused slides
